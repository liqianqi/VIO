# IMU 姿态 EKF 的 Yaw 漂移分析

> 对象:`test/test_imu.cpp` 中的 6 维四元数姿态 EKF
> 状态向量:`x_ = [q0, q1, q2, q3, bx, by]`(四元数 + 陀螺 x/y 轴零偏)
> 量测:仅加速度计(重力方向)

## 结论速览

- **现象**:IMU 在 yaw 方向剧烈晃动两下后,yaw 误差可达 ~16°,且事后无法自行恢复。
- **根因**:在"陀螺 + 加速度计"这套量测下,**yaw(绕重力轴的偏航)在数学上不可观测**。
  - yaw 由 `predict` 中 `wz` 的纯积分产生,任何误差直接累加。
  - `update` 中的量测模型 `x_h` 与雅可比 `H` 在结构上对 yaw 完全"盲",加速度计信息进不到 yaw 自由度。
- **能做的**:只能在"制造误差"一侧减缓(提高输出频率、精确积分、标定 z 轴),或在完整 VIO 中让**相机**来观测并修正 yaw。磁力计因电机磁场干扰不可用。

---

## A. yaw 是怎么产生的 —— 全在 `predict` 里积分出来

```cpp
// test/test_imu.cpp  predict()
double cx = wx - x_(4);
double cy = wy - x_(5);
double cz = wz;                 // z 轴角速度原样进积分(连零偏都没减)
...
Eigen::Vector4d q_new = q + 0.5 * omega_q * q * dt_;   // 一阶欧拉积分
```

- `cz = wz`:z 轴角速度**原样**进入积分——状态里只有 `bx, by`,没有 `bz`,所以 z 轴零偏无法扣除。
- `q_new = q + 0.5·Ω·q·dt`:yaw 完全是 `∫ wz dt` 的**航位推算(dead reckoning)**。

因此只要 `wz` 有任何误差(噪声、`bz` 零偏、高转速下的刻度因子误差、低采样率 + 一阶积分的离散误差),都会**直接积分累加进 yaw**。剧烈晃动时 `wz` 很大,离散积分误差被放大,于是攒出 16°。这一步只"制造" yaw,不含任何纠错。

---

## B. 加速度量测对 yaw 完全"看不见" —— `x_h` 与 yaw 无关

```cpp
// test/test_imu.cpp  update()
// 量测预测: 世界系"上"方向 [0,0,1] 旋到机体系 = R(q)^T * [0,0,1]
Eigen::Vector3d x_h;
x_h << 2 * (q[1] * q[3] - q[0] * q[2]),
       2 * (q[2] * q[3] + q[0] * q[1]),
       1 - 2 * (q[1] * q[1] + q[2] * q[2]);
Eigen::Vector3d y = z - x_h;    // 残差
```

`x_h` 是"重力方向在机体系下的预测"。给它代入一个**纯 yaw** 姿态:纯 yaw 转 ψ 的四元数为
`q = [cos(ψ/2), 0, 0, sin(ψ/2)]`,即 `q0=cos, q1=0, q2=0, q3=sin`。代入:

- `x_h[0] = 2(q1·q3 − q0·q2) = 2(0 − 0) = 0`
- `x_h[1] = 2(q2·q3 + q0·q1) = 0`
- `x_h[2] = 1 − 2(q1² + q2²) = 1`

得到 `x_h = [0, 0, 1]`,**与 ψ 完全无关**。即:不管 yaw 转到哪,残差 `y = z − x_h` 都一模一样。**量测中不含任何 yaw 信息。**

一般性证明:`x_h = R(q)^T·[0,0,1]`,而 yaw 是绕世界 z 轴的旋转 `R_z(ψ)`,满足 `R_z(ψ)^T·[0,0,1] = [0,0,1]`(绕 z 轴旋转不改变 z 轴),所以对任意 yaw,`x_h` 恒定。

---

## C. 为什么卡尔曼增益修不了 yaw —— yaw 落在 `H` 的零空间

```cpp
// test/test_imu.cpp  update()
Eigen::MatrixXd H = Eigen::MatrixXd::Zero(3, 6);
H.block<3,4>(0,0) <<
    -2 * q[2],  2 * q[3], -2 * q[0],  2 * q[1],
     2 * q[1],  2 * q[0],  2 * q[3],  2 * q[2],
            0, -4 * q[1], -4 * q[2],         0;
...
K_ = P_ * H.transpose() * S.inverse();   // 6x3
x_ = x_ + K_ * y;
```

同样代入纯 yaw 的 `q0=c, q1=0, q2=0, q3=s`,把 `H` 前 4 列算出来:

- 行 0:`[−2q2, 2q3, −2q0, 2q1] = [0,  2s, −2c, 0]`
- 行 1:`[ 2q1, 2q0,  2q3, 2q2] = [0,  2c,  2s, 0]`
- 行 2:`[0, −4q1, −4q2, 0]     = [0,   0,   0, 0]`

关键:**`q0` 列与 `q3` 列全为 0**。而 yaw 方向在四元数空间的扰动是
`∂q/∂ψ = ½·[−s, 0, 0, c]`,只落在 `q0、q3` 两个分量上。于是

```
H · (∂q/∂ψ) = 0
```

yaw 扰动正好在 `H` 的**零空间**中。`K = P·Hᵀ·S⁻¹` 带了 `Hᵀ`,所以增益 `K` 在 yaw 方向的分量为 0;`x_ += K·y` 因此**永远不会修正 yaw**。

---

## D. 协方差视角:yaw 的不确定度只增不减

```cpp
// predict()
P_ = F_ * P_ * F_.transpose() + Q_;

// update()
P_ = (Eigen::MatrixXd::Identity(6, 6) - K_ * H) * P_;
```

- **predict**:每步把 `Q_` 加进去,yaw 对应的方差**持续增大**。
- **update**:因为 `H` 对 yaw 灵敏度为 0,`(I − K·H)` 在 yaw 方向是恒等,**不削减 yaw 方差**。

结果:yaw 方差单调增长——滤波器"知道"自己对 yaw 越来越没把握,但没有任何量测能拉回来。这正是**不可观(unobservable)**在数值上的表现。

---

## 定位汇总

| 现象 | 代码位置(`test/test_imu.cpp`) |
|------|-------------------------------|
| yaw 由 `wz` 纯积分产生,误差直接累加 | `predict()`:`cz = wz`、`q_new = q + 0.5·Ω·q·dt` |
| `bz` 未建模,z 轴零偏无法扣除 | `predict()`:只减 `bx, by` |
| 量测 `x_h` 与 yaw 无关 → 残差不含 yaw 信息 | `update()`:`x_h` 三行公式 |
| `H` 对 yaw 灵敏度 = 0 → 增益 `K` 修不了 yaw | `update()`:`H` 与 `K_ = P·Hᵀ·S⁻¹` |
| yaw 方差只增不减(不可观) | `predict()`/`update()` 的 `P_` 更新 |

**一句话**:`predict` 负责把 `wz` 的误差积进 yaw,而 `update` 里 `x_h / H` 在结构上就对 yaw "盲",所以加速度计永远拉不回来。

---

## 缓解与解决方向

### 无法根治(需引入航向参考)

yaw 不可观是结构性的。只有引入一个**绝对航向参考**才能让 yaw 可观测:

- **磁力计**:本项目场景有电机,磁场干扰剧烈,**不可用**(板载 `0x59` 融合姿态内部也用磁力计,同样不可信)。
- **视觉(VIO 最终方案)**:相机观测到的特征点约束偏航,yaw 的可观测性由**相机**提供。纯 IMU 阶段 yaw 漂移属预期,交给视觉修正。

### 只能减缓(在"制造误差"一侧下手)

针对"剧烈晃动瞬间猛漂"这个具体症状,按性价比排序:

1. **提高输出频率 ODR + 波特率**(最对症):当前 9600 波特率下输出率偏低,快速转动时两帧间转角过大,一阶积分严重失真。提到 115200 / 100~200 Hz 通常能砍掉大半误差。
2. **精确积分代替一阶欧拉**:用指数映射 `q = q ⊗ exp(Δθ/2)`(`Δθ = ω·dt`),配合中点积分(相邻两帧角速度取均值)。
3. **陀螺标定**:静止求零偏并扣除;标定 z 轴**刻度因子**(高转速下按比例放大误差)。
4. **准确的 dt**:用设备固定 ODR 对应的 dt,避免主机时间戳抖动/延迟带来的积分误差。
5. **静止检测(ZUPT)**:静止时冻结 yaw 并重估零偏,防止长期累积。

> 说明:方法 3/4/5 主要治"零偏与慢漂";而"剧烈晃动瞬间猛漂"主要由**采样率不足 + 积分近似太糙**造成,故 **1 + 2 最对症**。
